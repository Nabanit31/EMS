# Employee Management System (EMS)

## 📌 Overview

The **Employee Management System (EMS)** is a modern web-based application built using **React.js** and **Tailwind CSS** that helps organizations efficiently manage employees and tasks. The application provides separate dashboards for **Admin** and **Employees**, enabling smooth workflow management, task assignment, and employee monitoring.

This project demonstrates practical implementation of:
- Authentication System
- Role-Based Dashboard Access
- Task Management
- State Management using Context API
- Local Storage Data Persistence
- Responsive UI Design

---

# 🚀 Features

## 🔐 Authentication System
- Secure Login for Admin and Employees
- Role-based access management
- Session persistence using Local Storage

---

## 👨‍💼 Admin Dashboard
Admin can:
- Manage employees
- Assign tasks
- Monitor employee activities
- Track completed and pending tasks
- Logout securely

---

## 👨‍💻 Employee Dashboard
Employees can:
- View assigned tasks
- Track task progress
- Monitor task completion status
- Access personalized dashboard

---

## 📋 Task Management
- Assign tasks to employees
- Track task status
- Manage workflow efficiently
- Task categorization and monitoring

---

## 🔔 Notification System
- Toast notifications using `react-toastify`
- Login alerts
- Logout notifications
- Error handling notifications

---

## 💾 Data Management
- Local Storage integration
- Persistent login support
- Efficient React state handling

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Nabanit31/EMS.git
```

---

## 2️⃣ Navigate to Project Directory

```bash

```

---

## 3️⃣ Install Dependencies

```bash

```

---

## 4️⃣ Run the Development Server
